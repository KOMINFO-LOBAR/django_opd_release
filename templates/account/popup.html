Petunjuk Pengisian

Fungsi Popup

Untuk menampilkan informasi berupa Info Grafis pada saat pertama kali pengunjung membuka website Anda.

Cara Mengisi Popup

  1. Klik tombol Tambah / Edit Data.
  2. Masukkan Judul Popup.
  3. Pilih Status (program akan otomatis mengaktifkan satu popup saja).
  4. Klik Choose File dan Crop Foto dengan dimensi 720 x 540 Pixel.
  5. Klik tombol Simpan.
  6. Refresh tampilan website Anda (Untuk menguji popup berjalan, tutup browser Anda, kemudian jalankan lagi).
Catatan :
Untuk hasil terbaik, masukkan gambar dengan resolusi 720 x 540 Pixel.
