Petunjuk Pengisian

Fungsi Dokumen

Menu Upload Dokumen ini sangat dibutuhkan di bagian Berita, Pengumuman, Artikel, maupun di Halaman Statis.
Dokumen Office seperti (doc, docx, xls, xlsx, ppt, pptx, pdf, jpg, jpeg, dll) dapat dimunculkan (di Preview) di bagian isi berita, pengumuman, artikel, atau halaman statis.

Selain itu Dokumen yang di Upload dapat diubah statusnya menjadi Publish agar muncul di menu Dokumen di halaman depan website Anda. Data di tabel tersebut sifatnya untuk umum, dan dapat di Download oleh pengunjung website Anda.

Cara Upload Dokumen

  1. Klik tombol Tambah Data / Edit Data.
  2. Input Nama Dokumen.
  3. Klik Choose File untuk mencari File Office atau Image di komputer Anda.
  4. Pilih Status.
    • Published: artinya dokumen akan langsung muncul di Menu Dokumen website.
    • Draft: artinya dokumen akan disimpan di database, dan tidak dimunculkan di menu dokumen website.
      Tetapi dapat di Preview di bagian isi Berita, Pengumuman, Artikel, maupun Halaman Statis.
  5. Klik tombol Simpan

Catatan :
Untuk digunakan sebagai Preview di menu lain, klik Copy URL to Clipboard di kolom Action tabel dokumen.
