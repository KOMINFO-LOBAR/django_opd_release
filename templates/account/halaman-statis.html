Fungsi Halaman Statis

Untuk menampilkan informasi dari menu yang baru dibuat oleh Admin website. Langkah yang di sarankan adalah:
  1. Buat Menu Baru terlebih dahulu.
    2. Abaikan langkah ini jika menu sudah ada, atau menggunakan menu Default yang sudah ada di website Anda.
  2. Buka tampilan depan website Anda.
  3. Klik Menu yang baru dibuat, atau menu yang telah ada di website Anda.
  4. Masuk ke Halaman Dashboard, pilih menu Halaman Statis.
  5. Klik tombol Edit di data dengan status Last Update terbaru.
  6. Lakukan Editing Data.
  7. Klik tombol Simpan.

Cara Mengisi Halaman Statis

  1. Klik tombol Tambah Data / Edit Data.
  2. Input judul Halaman Statis.
  3. Input Isi Halaman Statis.
  4. Editor di inputan isi Halaman Statis dapat menampung data seperti :
    • Teks / tulisan
    • Gambar, Upload Gambar Via Editor
    • Preview File Office, Preview Via Dokumen
    • Preview File Office dari Google Drive, Preview Via Google Drive
    • Buat Link ke Website Lain.
  5. Isi Akses Halaman Melalui Menu apa?
  6. Foto untuk slide show. Foto ini akan tampil di halaman depan website.
    • Foto berjumlah 3 buah, paling tidak ada 1 foto terisi agar memudahkan mengenali Halaman Statis di halaman depan website.
    • Crop Foto dengan dimensi 720 x 540 Pixel.
  7. Klik tombol Simpan
  8. Refresh tampilan website Anda.

Catatan :
File Office yang dapat di tampilkan preview nya adalah: pdf, doc, docx, xls, xlsx, ppt, pptx, png, jpg, dll.
