Petunjuk Pengisian

Fungsi Pengumuman

Untuk membuat pemberitahuan resmi tertulis mengenai kejadian tertentu yang sudah atau akan terjadi.
Seperti misalnya:
  • Pelayanan Selama Bulan Ramadhan Tetap Buka Pada Jam Kerja.
  • Gelombang I Penerimaan Bansos Dibuka Kembali Tanggal 1 Januari 2022.
  • Dan Lain-lain.
Informasi ini sifatnya hanya valid pada saat tertentu, admin dapat mengubah status pengumuman menjadi Draft jika informasi sudah tidak valid lagi. Data akan tetap tersimpan di database untuk keperluan arsip, atau untuk keperluan pembuatan pengumuman berikutnya.

Cara Mengisi Pengumuman

  1. Klik tombol Tambah Data / Edit Data.
  2. Input judul Pengumuman.
  3. Input Isi Pengumuman.
  4. Editor di inputan isi Pengumuman dapat menampung data seperti :
    • Teks / tulisan
    • Gambar, Upload Gambar Via Editor
    • Preview File Office, Preview Via Dokumen
    • Preview File Office dari Google Drive, Preview Via Google Drive
    • Buat Link ke Website Lain.
  5. Input status Pengumuman, Publish atau Draft.
  6. Foto untuk slide show. Foto ini akan tampil di halaman depan website.
    • Foto berjumlah 3 buah, paling tidak ada 1 foto terisi agar memudahkan mengenali Pengumuman di halaman depan website.
    • Crop Foto dengan dimensi 720 x 540 Pixel.
  7. Klik tombol Simpan
  8. Refresh tampilan website Anda.

Catatan :
File Office yang dapat di tampilkan preview nya adalah: pdf, doc, docx, xls, xlsx, ppt, pptx, png, jpg, dll.
