Petunjuk Pengisian

Fungsi Agenda

Untuk menampilkan kegiatan Instansi yang telah dilakukan atau yang akan dilakukan. Misalnya:
  1. Kegiatan Vaksinasi Balita Tanggal 1 September 2022.
  2. Kegiatan Bersepeda Dengan Door Prize Umroh Gratis.
  3. Dan Lain-lain.
Seluruh Informasi dapat dibuat Publish agar Instansi lain mengetahui kegiatan apa saja yang telah dan akan di lakukan.

Cara Mengisi Agenda

  1. Klik tombol Tambah Data / Edit Data.
  2. Input Nama Agenda.
  3. Input Deskripsi / Isi Agenda.
  4. Editor di inputan Deskripsi dapat menampung data seperti :
    • Teks / tulisan
    • Gambar, Upload Gambar Via Editor
    • Preview File Office, Preview Via Dokumen
    • Preview File Office dari Google Drive, Preview Via Google Drive
    • Buat Link ke Website Lain.
  5. Input Lokasi Kegiatan
  6. Input Tanggal Kegiatan
  7. Input Jam Kegiatan
  8. Input Penyelenggara Kegiatan (Jika menggunakan Event Organizer silahkan sebut namanya).
  9. Input Dihadiri Oleh Siapa?
  10. Input Status, Publish atau Draft.
  11. Klik tombol Simpan.
  12. Refresh tampilan website Anda.
