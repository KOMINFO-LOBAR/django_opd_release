1. Di Editor. Klik toolbar Gambar.
  2. Muncul window Image Properties.
  3. Klik tab Unggah.
  4. Klik Choose File untuk mencari file gambar yang ingin di upload.
  5. Klik Kirim ke server.
  6. Setelah berhasil dikirim ke server, tab Info Gambar otomatis akan aktif.
  7. Atur Lebar dan Tinggi gambar.
  8. Atur HSpace (Horizontal Space).
  9. Atur VSpace (Vertical Space).
  10. Atur Penjajaran Kiri atau Kanan).
  11. Klik OK jika sudah sesuai.
