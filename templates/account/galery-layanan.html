Fungsi Galeri Layanan

Sebagai landing page atau halaman utama yang pertama kali akan dilihat oleh pengunjung website Anda. Informasi yang di tampilkan dapat berupa Info Grafis atau Foto, seperti:
maklumat pelayanan, visi misi, jenis pelayanan, foto gedung instansi tampak depan, foto seluruh karyawan di depan gedung, dll.
Galery layanan akan muncul di bagian atas dari website Anda.

Cara Mengisi Galeri Layanan

  1. Klik tombol Tambah Data / Edit Data.
  2. Input judul foto.
  3. Pilih Status.
    • Published: artinya galery layanan akan langsung muncul di halaman depan website.
    • Draft: artinya galery layanan akan disimpan di database, dan tidak dimunculkan ke halaman depan.
      Jika ingin diubah statusnya, lakukan Edit data dan ubah status menjadi Published.
  4. Klik Pilih File untuk mencari file di komputer Anda.
  5. Crop Foto dengan dimensi 1055 x 550 Pixel.
  6. Klik tombol Simpan
  7. Refresh tampilan website Anda.
Catatan :
Untuk hasil terbaik, masukkan gambar dengan resolusi 1055 x 550 pixel.
