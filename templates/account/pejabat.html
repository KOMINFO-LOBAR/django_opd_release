Petunjuk Pengisian

Fungsi Pejabat

Untuk menyediakan informasi mengenai foto pejabat yang sedang aktif. Data Default berisi foto bupati dan wakil bupati, dapat dimunculkan dengan cara, klik tombol Refresh Data Pejabat di bagian kanan bawah Tabel Data Pejabat.

Data kepala instansi dapat diinput dengan cara klik tombol Tambah / Edit Data.

Cara Mengisi Pejabat

Jumlah data kepala instansi hanya satu untuk tiap-tiap instansi. Jika melakukan penginputan, maka data lama akan digantikan dengan data baru.
  1. Klik tombol Tambah / Edit Data.
  2. Masukkan nama lengkap kepala instansi.
  3. Masukkan jabatan kepala instansi.
  4. Klik Choose File untuk mencari foto terbaru kepala instansi di komputer Anda.
  5. Crop Foto dengan dimensi 200 x 250 pixel.
  6. Klik tombol simpan.
  7. Refresh tampilan website Anda.
Catatan :
Untuk hasil terbaik, masukkan gambar dengan resolusi 200 x 250 pixel.
