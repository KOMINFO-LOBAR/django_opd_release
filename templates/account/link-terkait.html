Fungsi Link Terkait

Untuk membuat link ke website lain yang berhubungan dengan Instansi Tertentu.
Copy URL dari website lain, dan paste di kolom inputan link di form ini untuk menambahkan link terkait.

Cara Mengisi Link Terkait

  1. Klik tombol Tambah Data / Edit Data.
  2. Input nama Link Terkait.
  3. Paste alamat website lain di kolom Link.
  4. Klik tombol Simpan.
  5. Refresh tampilan website Anda.
